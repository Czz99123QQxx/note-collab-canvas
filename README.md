# 🎵 Note Collab Canvas · 音符全球共创

> 像 r/place 一样，但用音符代替像素 —— 全球用户共同创作一首无限长的旋律。

🌐 **在线体验**：https://czz99123qqxx.github.io/note-collab-canvas

## ✨ 特性

- 🎹 **钢琴键盘** – 点击琴键选择音高（C3 ~ B6）
- 🧩 **协作网格** – 20x10 的共享画布，每人每次放置一个音符（或休止符）
- ⏸️ **休止符模式** – 一键切换，用于清除方格上的音符
- 🌍 **匿名共创** – 每次放置随机显示昵称，模拟全球用户实时协作
- ⏱️ **公平冷却** – 每位玩家放置后需等待 **90 秒**，无特权、无课金、人人平等
- 🌐 **8 种语言** – 简体中文、繁體中文、English、日本語、한국어、Français、Deutsch、Español
- ▶️ **全谱播放** – 按行播放整个网格（可调节播放速度：舒缓/适中/轻快）
- 📎 **导出乐谱** – 一键导出为纯文本乐谱
- 💾 **本地存储** – 自动保存画布状态，刷新不丢失

## 🎮 玩法说明

1. 点击钢琴键选择想要放置的音符（选中的键会高亮）
2. 点击画布上任意空白方格，即可放置该音符（同时会播放一声提示音）
3. 点击「休止符模式」可以切换到休止符，点击方格会清除原有音符
4. 每次放置后会有 90 秒冷却计时，倒计时结束才能再次放置
5. 点击「播放全谱」可以按当前速度顺序演奏整个画布（行优先）
6. 使用底部按钮控制播放/暂停/停止，拖拽进度条可以跳转播放位置
7. 右上角可切换界面语言

## 🛠️ 技术栈

- 原生 HTML5 / CSS3 / JavaScript (ES6)
- Web Audio API（播放音符）
- LocalStorage（持久化）
